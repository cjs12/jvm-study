## 2.1 安装CYGWIN http://www.cygwin.com/

注意： 不能默认安装，必须手工选择的安装项目有：

| 文件名 | 分类 | 包 | 
| -- | -- | -- |
| ar.exe | Devel | binutils |
| make.exe | Devel | make |
| m4.exe | Interpeters | m4 |
|cpio.exe | Utils | cpio |
| gawk.exe | Utils | awk |
| file.exe |Utils | file |
| zip.exe | Archive | zip |
| unzip.exe | Archive | unzip |
| free.exe | System | procps |
